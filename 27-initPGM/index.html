# initPGM 接口介绍

TIP

说明：初始化 pgm

options 参数：

名称 类型 是否必选 描述
addPgmData object Y pgm 参数
videoEl object Y dom Video
audioEl int Y dom Audio

pgm 参数:

名称

类型

是否必选

示例值

描述

method

string

Y

switcherAddPGM

系统规定参数，取值：switcherAddPGM

switcherId

string

Y


导播台Id

sessionIdstringY
会话Id
componentIdintN
组件Id
workChipintN0组件是否使用gpu处理，0:cpu,1:gpu
cardIndexintN
卡号

举例：

//放大
{
    "request": true,
    "id": 123456,
    "method": "initPGM",
    "data": {
            addPvwData: {
                switcherId: window.sessionStorage.getItem('switcherId'),
                roomId: window.sessionStorage.getItem('roomId'),
                type: 2003,
                width: 640,
                height: 360,
                bPreview: true,
                componentId: 0
            },
            videoEl: document.getElementById('pvwVideo'),
            audioEl: document.getElementById('pvwAudio'),
            inputId: 0
        }
}

 {
    "response":true,
    "id"      :123456,
    "ok"      :true,
    "data":{
        "switcherId": "hd-12345",
        "componentId": 0
    }
}

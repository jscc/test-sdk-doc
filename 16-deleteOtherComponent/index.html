# deleteOtherComponent 接口介绍

TIP

说明：删除组件

options 参数：

名称 类型 是否必选 描述
switcherId string Y 导播台 Id
componentId number Y cg 组件 Id

举例：

{
    "request": true,
    "id": 123456,
    "method": "deleteOtherComponent",
    "data": {
        "switcherId": "hd-12345",
        "componentId":0
    }
}

返回值:

 "response":true,
    "id"      :123456,
    "ok"      :true,
    "data":{
        "switcherId": "hd-12345"
    }

# addYbMeetingSwitcher

# addYbMeetingSwitcher 接口介绍

TIP

说明：添加医百会议输入源

options 参数：

名称 类型 是否必选 描述
addInputData object Y input 参数
addOutputData object Y output 参数
videoEl HTMLVideoElement Y 视频 dom
audioEl HTMLAudioElement Y 音频 dom
cgId number Y cg 组件 Id

addInputData 参数：

名称 类型 是否必选 描述
switcherId string Y 导播台 Id
name string Y 输入源名称，端上发
ybMeetNumber string Y 医百会议号
roomId string Y 100rtc,房间 Id
peerId string Y 100rtc,用户 Id
roleType number Y 医百会议输入源，流类型 普通参会者(normal):0, 发言方(speaker): 1,屏幕共享(share):2

addOutputData 参数：

名称 类型 是否必选 描述
switcherId string Y 导播台 Id
type int Y 输出类型，见下表
bPreview bool Y 是否是预览输出,100rtc 输出类型分两种：1.医百会议输出 false 2.和预览输出 true 医百会议号
roomId string Y 房间 Id
width string Y 输出流宽度，以像素为单位
height number Y 输出流高度，以像素为单位
componentId number Y 医百会议输入源，流类型 普通参会者(normal):0, 发言方(speaker): 1,屏幕共享(share):2

type 参数：

定义

值

描述

ePayload_StreamOut

2001

流

ePayload_FileOut

2002

文件

ePayload_100RtcOut

2003

100 rtc




举例：

await addYbMeetingSwitcher({
    addInputData: {
        switcherId,
        name: data.userName,
        type: data.type,
        ybMeetNumber: data.ybMeetNumber,
        roomId: data.roomId,
        peerId: data.peerId || data.userId,
        roleType: roleMap[data.ybMeetViewType]
    },
    addOutputData,
    videoEl,
    audioEl,
    cgId: data.cgId || 0
});

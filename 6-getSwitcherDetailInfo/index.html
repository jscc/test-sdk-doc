# getSwitcherDetailInfo

# getSwitcherDetailInfo 接口介绍

TIP

说明：获取导播台详细配置信息

options 参数：

名称 类型 是否必选 描述
switcherId string Y 导播台 Id

举例：

try {
    await getSwitcherDetailInfo({
        switcherId: 'hd-12345'
    });
} catch (error) {
    console.log(error);
}

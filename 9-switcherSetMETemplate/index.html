# switcherSetMETemplate 接口介绍

TIP

说明：更新布局配置

options 参数：

名称





类型

是否必选

示例值

描述

switcherId





string

Y


导播台Id

sessionId



stringY
会话Id
componentId



intY
组件Id
type



intY
me模版类型，0：固定模版，1：自定义模版 , 2:虚拟场景
backgroundUrl



stringY
背景图地址
templateThumbnail



stringY
模版缩略图
templateName



stringY
模版名称
templateId



stringY
固定模版Id
templateInfo



array

Y


模版内容

inputId


int

Y


key(regionId)对应的输入源组件Id

inputType


intY

0: 普通输入源

1:  图片（虚拟场景）


pictureAddress


stringN
如果inputType是1，为图片地址

region


objectY
输入源组件对于的绘制区域


regionId

stringY
混流layer,0、1、2、3...累加


shape

string

Y


区域形状类型：

rectangle:矩形   circle: 圆形   heart: 心形  ellipse: 椭圆 diamond: 菱形   stretch:梯形  pillow:枕型曲面



shapeId

intY
区域形状Id

0:矩形   1: 圆形   2: 心形   3: 椭圆   4: 菱形   5:梯形 6: 枕型曲面



path

arrayY

前端需要字段，在恢复detail接口返回给前端

path : [ { x:0, y:0}, { x:100, y:0}]




x
intY



y
intY


area

objectY
位置信息，left,top,width,heigth



left
objectN

矩形/椭圆/菱形/心形/圆形：左上角距离左边界，

比例：numerator / denominator




top
objectN

矩形/椭圆/菱形/心形/圆形：左上角距离上边界，

比例：numerator / denominator




width
objectN

矩形/椭圆/菱形：宽，

比例：numerator / denominator




height
objectN

矩形/椭圆/菱形/心形/圆形：高，

比例：numerator / denominator













upperLeftX
objectN
梯形:左上角X



upperLeftY
objectN
梯形:左上角Y



upperRightX
objectN
梯形:右上角X



upperRightY
objectN
梯形:右上角Y



bottomLeftX
objectN
梯形:左下角X



bottomLeftY

bottomRightX
